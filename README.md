# Coach Matthews | Performance Specialist Landing Page & Data Pipeline

A production-ready, full-stack web application designed for elite fitness coaching analytics. This platform features a dynamic frontend interface built with Streamlit, an integrated metabolic/macro engineering calculation engine, and a secure serverless background data pipeline that streams client intake metrics straight into a cloud analytics database.
