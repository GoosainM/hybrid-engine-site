## 🛡️ Project 2: Multi-Threaded Network Port Scanner
A high-performance command-line interface (CLI) cybersecurity utility engineered to audit network parameters and identify open listening sockets during infrastructure security reviews.

### ⚙️ Technical Highlights (Project 2)
* **Asynchronous Execution:** Implements Python's native `concurrent.futures.ThreadPoolExecutor` to manage a concurrent thread matrix, dropping overall scan times from minutes to seconds by processing socket connections simultaneously.
* **Low-Level Socket Integration:** Utilizes raw `socket.socket(socket.AF_INET, socket.SOCK_STREAM)` connections to attempt non-blocking TCP handshakes (`connect_ex`) against target host interfaces.
* **Fault-Tolerant Network Layer:** Features automatic hostname resolution (`socket.gethostbyname`) with built-in network layer parameter handling (`socket.gaierror`) to gracefully manage offline target endpoints.
